### 엘리먼트 프로퍼티

각 선택 가능 엘리먼트는 다음 프로퍼티를 가집니다:

```javascript
{
    // 식별
    type: 'headline' | 'subtext' | 'cta',
    id: 'ad-headline',
    
    // 위치
    x: 540,          // 왼쪽에서 px
    y: 300,          // 위에서 px
    width: 800,      // px
    height: 'auto',  // px 또는 'auto'
    rotation: 0,     // 도
    
    // 타이포그래피
    fontFamily: 'Pretendard',
    fontSize: 64,    // px
    fontWeight: 700,
    textAlign: 'center',
    letterSpacing: 0, // em
    lineHeight: 1.2,  // 비율
    color: '#FFFFFF',
    
    // 가시성
    visible: true,
    locked: false
}
```

### 배경 격리 규칙

**핵심 설계 결정:** 배경(ad-container, ad-overlay, 배경 이미지)은 일반 클릭 인터랙션을 통해 절대 선택되지 않습니다.

```javascript
// 선택 가드
if (target.classList.contains('ad-container') ||
    target.classList.contains('ad-overlay') ||
    target.classList.contains('ad-background-image')) {
    // 선택하지 않음 - 배경 엘리먼트임
    return;
}
```

이는 사용자가 빈 캔버스 영역을 클릭하려 할 때 실수로 배경을 선택하는 것을 방지합니다.
