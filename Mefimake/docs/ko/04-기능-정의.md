# 기능 정의

이 문서는 MEFIMAKE의 모든 주요 기능에 대한 상세 명세를 제공합니다.

---

## 1. AI 콘텐츠 생성

### 목적
사용자가 처음부터 시작하지 않고 여러 카피 변형을 빠르게 생성할 수 있게 하여, 크리에이티브 블록을 줄이고 아이디어 발상을 가속화합니다.

### 로직

```
입력: 키워드 (예: "자격증")
    │
    ▼
┌─────────────────────────┐
│   콘텐츠 생성           │
│   알고리즘              │
│                         │
│   - 5개 사전 정의된     │
│     변형 템플릿         │
│   - 키워드 삽입         │
│   - 스타일 변형         │
└───────────┬─────────────┘
            │
            ▼
출력: 5개 고유 조합
    ├── 변형 1: 헤드라인 + 서브텍스트 + CTA
    ├── 변형 2: 헤드라인 + 서브텍스트 + CTA
    ├── 변형 3: 헤드라인 + 서브텍스트 + CTA
    ├── 변형 4: 헤드라인 + 서브텍스트 + CTA
    └── 변형 5: 헤드라인 + 서브텍스트 + CTA
```

### 데이터 구조

```javascript
const contentVariants = [
    {
        headline: `${keyword} 취득의 새로운 기준`,
        subtext: `전문가가 설계한 커리큘럼으로 빠르게 합격하세요`,
        cta: '무료 상담받기'
    },
    {
        headline: `왜 ${keyword}인가?`,
        subtext: `높은 연봉, 안정적인 미래가 기다립니다`,
        cta: '자세히 보기'
    },
    // ... 3개 더
];
```

### 사용자 플로우
1. 사용자가 입력 필드에 키워드 입력
2. "AI 초안 문구 생성" 버튼 클릭
3. 시스템이 5개 변형을 생성하고 순환
4. 각 클릭마다 다음 변형 표시
5. 변형 5 이후 변형 1로 순환

### 제한사항
- 현재는 사전 정의된 템플릿 사용 (실제 AI 아님)
- 템플릿은 한국어만 지원
- 키워드당 5개 변형으로 제한
- 사용자 선호도 학습 없음

## 2. 캔버스 시스템

### 목적
최종 내보내기 광고를 정확하게 나타내는 WYSIWYG 편집 표면 제공.

### 지원 사이즈

| 이름 | 화면비 | 크기 | 사용처 |
|------|--------|------|--------|
| Square | 1:1 | 1080×1080 | Instagram 피드 |
| Landscape | 16:9 | 1280×720 | Facebook 피드 |
| Portrait | 9:16 | 1080×1920 | Instagram 스토리 |

### 데이터 구조

```javascript
const canvasState = {
    width: 1080,
    height: 1080,
    aspectRatio: '1:1',
    scale: 0.5,  // 표시 스케일 (반응형)
    elements: [
        { type: 'headline', ... },
        { type: 'subtext', ... },
        { type: 'cta', ... }
    ]
};
```

### 제한사항
- 줌/팬 컨트롤 없음
- 고정된 엘리먼트 유형 (헤드라인, 서브텍스트, CTA)
- 임의 도형 그리기 없음
- 세션당 단일 캔버스

## 3. 선택 엔진

### 목적
시각적 피드백과 프로퍼티 패널 동기화와 함께 직관적인 엘리먼트 선택 가능.

### 시각적 피드백

| 상태 | 외관 |
|------|------|
| 일반 | 아웃라인 없음 |
| 호버 | 미세한 하이라이트 |
| 선택됨 | 파란색 아웃라인 (2px solid #0A84FF) |
| 드래그 중 | 투명도 감소 |

### 제한사항
- 단일 선택만 (멀티 선택 없음)
- 키보드 기반 선택 없음
- 선택 히스토리 없음 (선택 실행취소)

---

## 4. 배경 컨트롤

### 목적
단색, 그라데이션, 효과가 있는 이미지를 포함한 풍부한 배경 커스터마이징 가능.

### 로직

```
배경 유형 선택
    │
    ├── 단색 ──────► 단일 컬러 피커
    │               └── ad-container에 적용
    │
    ├── 그라데이션 ──► 시작/끝 컬러 피커
    │               └── 각도 컨트롤
    │               └── linear-gradient 적용
    │
    └── 이미지 ────► 파일 업로드
                    └── 핏 모드 (cover/contain)
                    └── 불투명도 슬라이더
                    └── 블러 슬라이더
```

### 데이터 구조

```javascript
const backgroundState = {
    type: 'gradient',  // 'solid' | 'gradient' | 'image'
    
    // 단색
    solidColor: '#00298A',
    
    // 그라데이션
    gradientStart: '#001D61',
    gradientEnd: '#003C8A',
    gradientAngle: 180,
    
    // 이미지
    imageUrl: null,
    imageFit: 'cover',
    imageOpacity: 100,
    imageBlur: 0
};
```

### 제한사항
- 이미지 포지셔닝 컨트롤 없음
- 단일 이미지만 (레이어 없음)
- 비디오 배경 없음
- 로컬 이미지만 (URL 입력 없음)

## 5. 타이포그래피 시스템

### 목적
광고 카피를 위한 전문가급 타이포그래피 컨트롤 제공.

### 사용 가능한 컨트롤

| 컨트롤 | 프로퍼티 | 범위/옵션 |
|--------|----------|-----------|
| 폰트 패밀리 | fontFamily | 6개 폰트 옵션 |
| 폰트 크기 | fontSize | 8 - 200px |
| 폰트 굵기 | fontWeight | 300 - 900 |
| 텍스트 정렬 | textAlign | left, center, right |
| 자간 | letterSpacing | -0.1 - 0.5em |
| 행간 | lineHeight | 80% - 200% |

### 폰트 스택

```css
--fonts: 
    'Pretendard',    /* 한국어 주 폰트 */
    'Inter',         /* 라틴 주 폰트 */
    'Noto Sans KR',  /* 한국어 폴백 */
    'Space Grotesk', /* 디스플레이 */
    'Syne',          /* 헤드라인 */
    sans-serif;
```

### 제한사항
- 커스텀 폰트 업로드 없음
- 텍스트 효과 없음 (그림자, 아웃라인)
- 패스 위 텍스트 없음
- 고정 텍스트 박스 (자동 리사이즈 없음)

---

## 6. 레이어 시스템

### 목적
엘리먼트 스태킹 순서의 시각적 관리 가능.

### 기능

| 기능 | 설명 |
|------|------|
| 순서 변경 | 드래그하여 z-index 변경 |
| 가시성 | 눈 아이콘으로 표시/숨김 |
| 잠금 | 잠금 아이콘으로 편집 방지 |
| 이름 변경 | (구현되지 않음) |

### 제한사항
- 고정 3개 레이어 (추가/삭제 불가)
- 레이어 그룹 없음
- 레이어 효과 없음
- 레이어 복제 없음

---

## 7. 내보내기 시스템

### 목적
여러 포맷과 사이즈로 제작 준비된 이미지 에셋 생성.

### 내보내기 모드

| 모드 | 출력 | 사용처 |
|------|------|--------|
| 현재 이미지 | 1 PNG | 빠른 단일 내보내기 |
| 사이즈 세트 | 3 PNG | 멀티 플랫폼 캠페인 |
| 전체 변형 | 15 PNG | A/B 테스트 세트 |

### 파일 명명 규칙

```
{키워드}-{변형}-{사이즈}.png

예시:
- 자격증-v1-square.png
- 자격증-v1-landscape.png
- 자격증-v2-portrait.png
```

### 제한사항
- PNG만 (JPG, WebP 없음)
- 품질 설정 없음
- 배치 ZIP 다운로드 없음
- 순차 다운로드 (병렬 아님)

---

## 8. 히스토리 시스템

### 목적
사용자가 작업 상태를 저장하고 이전 버전을 복원할 수 있게 함.

### 기능

| 기능 | 설명 |
|------|------|
| 저장 | 썸네일과 함께 현재 상태 캡처 |
| 복원 | 저장된 상태를 에디터에 로드 |
| 삭제 | 히스토리에서 항목 제거 |
| 미리보기 | 히스토리 패널의 썸네일 |

### 제한사항
- 로컬 스토리지만 (클라우드 없음)
- ~5MB 스토리지 제한
- 버전 비교 없음
- 선택적 복원 없음 (전체 또는 없음)

---

## 9. Safe Zone 및 시장 조사

### 목적
사용자가 플랫폼 규정 준수 광고를 만들고 경쟁사를 조사하도록 지원.

### Safe Zone

```
┌──────────────────────────┐
│ ┌──────────────────────┐ │
│ │                      │ │
│ │    SAFE ZONE         │ │
│ │    (콘텐츠 영역)     │ │
│ │                      │ │
│ └──────────────────────┘ │
│     UNSAFE (잘릴 수 있음)│
└──────────────────────────┘
```

토글로 플랫폼별 안전 영역을 나타내는 오버레이 표시/숨김.

### 시장 조사 링크

| 버튼 | 목적지 | 목적 |
|------|--------|------|
| Safe Zone | 오버레이 토글 | 플랫폼 규정 준수 |
| Meta Library | facebook.com/ads/library | 경쟁사 조사 |
| WebScout Agent | webscout-agent.vercel.app | 시장 분석 |

### 제한사항
- 일반 safe zone (플랫폼별 아님)
- 외부 링크만 (통합 없음)
- 저장된 조사 노트 없음
