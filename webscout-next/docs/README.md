## WebScout Next — v1 First Pass

This `docs` folder describes the **first UX/UI and feature hardening pass** ("1차 고도화") for the WebScout Next.js application.

- `01-product-overview.md` – what WebScout is and who it is for  
- `02-ux-ui-first-pass.md` – homepage, loading, sitemap UX, and interactions  
- `03-insights-and-scoring.md` – Insight Score, SEO, and UX / Structure scoring model  
- `04-tech-and-security.md` – architecture notes, API proxy, and security hardening  
- `05-next-steps.md` – suggested roadmap items for future paid subscription  
- `06-link-graph-and-crawler.md` – link graph design intent, graph-friendly crawler, and future features (key pages, orphans, PageRank-style scoring)

For a deeper, low-level description of the implementation, see the root-level `WEBSCOUT_AUDIT.md`.
