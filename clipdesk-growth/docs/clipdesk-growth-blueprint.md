# Clipdesk Growth Blueprint

**하반기 전략 설계 — 방향 고정용 ⭕ (실행 ❌)**

---

## 1. Clipdesk 정의

- **Clipdesk**: AI 기반 콘텐츠 자동화 플랫폼 + 전문가 하이브리드 제작
- Textar(국내) + Textar Global(해외) 통합 브랜드
- 단순 자막 대행·번역 아님 → 품질·비용·속도를 동시에 만족시키는 하이브리드 업무 자동화

---

## 2. 타겟 세그먼트

| 세그먼트 | Clipdesk 가치 |
|----------|---------------|
| 기업 홍보·마케팅팀 | 반복 영상 자동 자막·번역, 제작 속도 확보 |
| 미디어·콘텐츠 제작사 | 대량 자동화로 원가 절감·빠른 납품 |
| 글로벌 진출 기업 | 다국어·현지화 패키지 |

---

## 3. Core Use Case 3개 (기능 기준)

1. **실시간 AI 자막 기록** (clipdesk-caption) — 녹화·스트리밍 시 실시간 자막 자동화
2. **영상 자막 다국어 번역** (clipdesk-translate) — 다국어 배포·현지화
3. **AI 기반 영상 편집** (clipdesk-edit) — 자동 편집·패키징

---

## 4. timblo 연계 구조

```
회의·음성     →  timblo   (회의록, 요약, 인사이트)
영상·콘텐츠·언어  →  Clipdesk (자막, 번역, 패키징)
```

TIMBEL 산하 커뮤니케이션 자동화 제품.

---

## 5. 하반기 Phase A/B/C

| Phase | 목적 | 산출물 |
|-------|------|--------|
| **Phase A** | 브랜드 통합 | Clipdesk 가이드, 메시지, 제품 명칭 |
| **Phase B** | 랜딩 + 사례 구축 | 리뉴얼 랜딩, 타겟별 사례, 문의 유입 |
| **Phase C** | Hybrid 마케팅 시작 | 세그먼트별 콘텐츠, 채널 운영 |

※ **Phase + Channel 병행**: Phase는 언제 무엇을 하느냐, Channel은 어디서 계속 움직이느냐. [channel-ia.md](channel-ia.md) 참조.
