## 시나리오 개요

| 항목 | 내용 |
|------|------|
| 시나리오명 | 브랜드 언급 모니터링 |
| 트리거 | 새 Google Alert 감지 시 |
| 출력 | Notion Database |

---

## 모듈 구성

```
[RSS] Watch RSS feed items (Google Alerts)
    ↓
[Notion] Create a Database Item
```

---

## Google Alerts 설정

### 알림 생성 방법

1. https://www.google.com/alerts 접속
2. 검색어 입력
3. 옵션 설정:
   - 빈도: 수시로
   - 소스: 자동
   - 언어: 한국어
   - 지역: 대한민국
   - 개수: 모든 결과
   - **전송 위치: RSS 피드** ← 중요!
4. 알림 만들기 클릭
5. RSS 아이콘 우클릭 → 링크 복사

## 트러블슈팅

### 문제: RSS에서 데이터 없음
- **원인:** Google Alerts에 아직 새 결과 없음
- **해결:** 정상! 새 알림 생기면 자동 수집됨

### 문제: 테스트 필요
- **해결:** 임시로 바이라인 RSS로 변경해서 테스트 후, 다시 Google Alerts RSS로 변경

## 활용 방안

### 1. 자사 언급 모니터링
- 팀블로, 팀벨 언급 기사 수집

### 3. 시장 트렌드

- 블로그 소재 발굴
