# timblo Growth Operating System

**초기 스테이지 B2B AI SaaS를 위한 공개 성장 운영 시스템**

---

## 이 레포가 하는 일

마케팅 문서가 아닙니다. timblo의 **리베뉴 운영 시스템(Revenue Operating System)** 과 개인 작업 방법론입니다. 3인 Revenue Squad가 운영하는 초기 스테이지 B2B AI 회의 솔루션입니다.

전략, 일일 실행, 자동화 아키텍처를 공개 문서로 남겨서:
- 팀과 이해관계자 정렬
- 열린 학습
- 반복 가능한 성장 모델 구축

---

## 성장 철학

**timblo 마케팅은 채널 운영이 아닙니다.**

**실제 업무 자동화 사례의 축적** → 리드 → 미팅 → 계약입니다.

리치보다 증거를 우선합니다. 모든 콘텐츠는 이 질문에 답합니다: *"어떤 실제 업무를 자동화했고, 누구를 도왔는가?"*

---

## North Star

모든 활동이 이 루프에 연결됩니다.

---

## 리베뉴 루프

| 단계 | 입력 | 산출 |
|------|------|------|
| 1. 유입 | 콘텐츠, SNS, SEO, 유료 | 인지, 트래픽 |
| 2. 전환 | GEO 랜딩, 블로그, 데모 CTA | 데모 요청 |
| 3. 자격화 | 영업 대화 | 미팅 |
| 4. 마감 | 미팅, 제안 | 계약 |

---

## Phase 로드맵

| Phase | 기간 | 모드 | 목표 |
|-------|------|------|------|
| **Phase 1** | 0~2개월 | 수동 | 메시지 검증, 데모 유도 요인 학습 |
| **Phase 2** | 2~4개월 | 하이브리드 | 수동 콘텐츠 + 리드 자동 수집 |
| **Phase 3** | 4개월+ | 반자동 | 스코어링·인수인계 기반 성장 엔진 |

---

## 팀 구성

| 역할 | 담당 | 포커스 |
|------|------|--------|
| **총괄** | 성장 오너 | 퍼널, 메시지, 자동화 |
| **영업** | 리드 인테이크 | 24시간 응답, 시장 피드백 |
| **디자이너** | 전환 | 랜딩, CTA, 콘텐츠 비주얼 |

## 블로그 배너 브랜딩

**자사 블로그 포스팅용 배너 시안**입니다. 포스트 하단에 팀 소개 배너를 삽입할 때 사용합니다.

| 항목 | 내용 |
|------|------|
| **메인** | 실무자가 직접 만드는 AI 솔루션 timblo |
| **서브** | 자사 블로그 포스팅용 배너 시안 브랜딩 |

**팀 카드 순서**: 이세영(Head of Growth) → 박한주(Business Consultant) → 서동숙(Visual Designer) → 길은진(UX/UI Designer)

→ [배너 시안 보기](docs/timblo_팀_배너.html)
