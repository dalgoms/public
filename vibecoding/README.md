## Why it matters | 왜 의미가 있나요

**EN**

I wanted to test one simple idea:

> Can someone without a traditional engineering background  
> turn ideas into working products with AI?

The answer was **yes**.

This project proved that:
- Ideas can become interfaces
- Conversations can become code
- AI can act as a real builder, not just a chatbot

**KR**

제가 확인하고 싶었던 건 딱 하나였습니다:

> 개발자가 아니어도  
> AI와 함께 실제 작동하는 제품을 만들 수 있을까?

답은 **Yes**였습니다.

이 프로젝트를 통해 확인한 것들:
- 아이디어는 곧 UI가 되고
- 대화는 코드가 되며
- AI는 단순 챗봇이 아니라 "빌더"가 될 수 있다는 것

## About Vibe Coding | Vibe Coding이란

**EN**

Building through conversation with AI.

1. Describe intent
2. AI generates implementation
3. Human refines direction
4. Repeat

It's not about perfect code.  
It's about momentum.

**KR**

AI와 대화하며 만드는 방식입니다.

1. 의도를 말로 설명
2. AI가 코드 생성
3. 사람이 방향 조정
4. 반복

완벽한 코드보다  
속도와 흐름이 중요합니다.
