# Clipdesk Growth System

Clipdesk는 Textar + Textar Global을 통합한 **AI 기반 콘텐츠 자동화 플랫폼**입니다.

본 레포지토리는 Clipdesk의 하반기 Growth 전략 + KOBA 실행 + 리드 운영 과정을 기록합니다.

---

## 목표

- KOBA 리드 확보
- 미팅 전환
- 계약 연결

---

## 구성

| 폴더 | 용도 |
|------|------|
| **docs** | 전략 및 운영 설계 |
| **logs** | 일자별 실행 기록 |
| **operations** | 전시 + 주간 운영 |
| **automation** | 리드 자동화 구조 |
| **templates** | 이메일/영업 템플릿 |
| **forms** | 리드 폼 설계 |
| **assets** | 스크린샷 등 |

---

## 프레임

- **timblo** = Revenue OS (실행 중)
- **clipdesk** = Growth OS + KOBA 런치

둘 다: 설계 → 운영 → 로그 → 자동화
