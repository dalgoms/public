# 제품 개요

## MEFIMAKE란?

MEFIMAKE는 퍼포먼스 마케팅 팀을 위해 설계된 브라우저 기반 크리에이티브 IDE입니다. 디자인 도구와 광고 제작 워크플로우 사이의 간극을 메우며, 고전환율 광고 크리에이티브의 빠른 제작과 반복을 가능하게 합니다.

---

## 우리가 해결하는 문제

### 광고 크리에이티브 제작의 현실

퍼포먼스 마케팅 팀은 근본적인 병목에 직면해 있습니다: **크리에이티브 속도**.

**문제점:**

1. **반복적인 수작업**
   - 디자이너는 80%의 시간을 포맷팅과 리사이징에 소비
   - 같은 콘텐츠, 다른 사이즈 = 수 시간의 작업
   - 디자인 도구 간 복사-붙여넣기 워크플로우

2. **제한된 A/B 테스트 역량**
   - 테스트에는 여러 변형이 필요
   - 제작 역량이 실험을 제한
   - 승리하는 크리에이티브 패턴을 너무 늦게 발견

3. **파편화된 워크플로우**
   - 카피는 구글 독스에
   - 디자인은 피그마/캔바에서
   - 내보내기는 수동 리사이징 필요
   - 통합된 크리에이티브 제작 환경 부재

4. **도구 불일치**
   - Figma: 빠른 광고 반복에는 너무 복잡
   - Canva: 전문적인 컨트롤에는 너무 단순
   - 둘 다 광고 전용 워크플로우에 최적화되지 않음

---

## 우리의 솔루션

MEFIMAKE는 광고 제작을 이해하는 목적에 맞게 구축된 크리에이티브 IDE입니다.

### 핵심 원칙

1. **완벽함보다 속도**
   - 반복 속도에 최적화
   - 1개를 완벽하게 만드는 시간에 10개의 변형 제작

2. **AI 지원 콘텐츠**
   - 카피 변형을 즉시 생성
   - 빈 페이지에서 시작하지 않음

3. **멀티 포맷 우선**
   - 하나의 디자인, 여러 출력
   - 자동 화면비 적응

4. **통합된 워크플로우**
   - 기획 → 디자인 → 내보내기를 하나의 인터페이스에서
   - 컨텍스트 전환 없음

## 타겟 사용자

### 1차: 퍼포먼스 마케터
- Meta, Google, TikTok에서 유료 캠페인 운영
- 대량의 크리에이티브 제작 필요
- 데이터 기반 크리에이티브 결정

### 2차: 소규모 크리에이티브 팀
- 전담 디자이너 없는 1-3인 팀
- 디자인 배경 없이 빠르게 움직여야 함
- 기능 수보다 단순함을 중시

### 3차: 프리랜스 디자이너
- 비슷한 니즈를 가진 여러 클라이언트 담당
- 템플릿과 속도 필요
- 시간을 늘리지 않고 산출물 확대 원함

## 제품 철학

### 적은 것이 많은 것
우리는 Figma가 되려 하지 않습니다. 하나의 문제를 극도로 잘 해결합니다: **광고 크리에이티브를 아이디어에서 내보내기까지 가능한 빠르게**.

### 의견이 있는 기본값
무한한 옵션 대신, 90%의 사용 사례에 작동하는 스마트한 기본값을 제공합니다. 파워 유저는 커스터마이징할 수 있지만, 초보자는 즉시 출시할 수 있습니다.

### 부조종사로서의 AI
AI는 대체가 아닌 보조입니다. 인간이 다듬을 시작점을 생성합니다. 최종 크리에이티브 결정은 항상 사용자의 것입니다.

## 성공 지표

MEFIMAKE가 작동하는지 측정하는 방법:

1. **첫 내보내기까지의 시간** - 새 사용자가 첫 광고를 얼마나 빨리 제작할 수 있는가?
2. **세션당 변형 수** - 세션당 몇 개의 크리에이티브 변형이 제작되는가?
3. **재사용률** - 사용자가 다음 캠페인을 위해 돌아오는가?
4. **내보내기 완료율** - 시작된 프로젝트 중 몇 %가 내보내기되는가?

## 비전

**1단계: 크리에이티브 IDE** (현재)
빠른 크리에이티브 제작을 위한 독립형 도구.

**2단계: 크리에이티브 인텔리전스**
크리에이티브 성과 패턴에 대한 분석과 인사이트.

**3단계: 크리에이티브 플랫폼**
팀 협업, 에셋 관리, 직접 광고 플랫폼 연동.

## 요약

MEFIMAKE는 퍼포먼스 마케터들이 자신의 워크플로우에 맞게 구축된 크리에이티브 도구를 가질 자격이 있기 때문에 존재합니다 — 적응해야 하는 범용 디자인 도구가 아닌.

우리는 가장 강력한 디자인 도구를 만들고 있지 않습니다. 크리에이티브 브리프에서 라이브 광고까지의 가장 빠른 경로를 만들고 있습니다.
